Streamline your network with smart automation: connect TP-Link Easy Smart Switch devices to Homey for port control, status, and Flow-driven network automation.

Features:
- Port Control: Enable or disable individual ports on your TP-Link Easy Smart Switch directly from the Homey app or through Homey Flows.
- Device Integration: Seamlessly integrate with multiple TP-Link Easy Smart Switch models.
- Automation: Create Homey Flows to automate network management tasks based on your preferences or specific conditions—including LED control, reacting when link state changes, and restarting the switch when needed.
- Status Monitoring: View the status of each port (enabled/disabled) within the Homey app.
- Detection of Link Changes: Switch ports are monitored for link status about every minute.

The switch must be on the same local network as your Homey. Use a fixed IP address or a DHCP reservation so the address does not change. Enter the same username and password you use for the switch’s web management page.

Supported hardware:

- TP-Link TL-SG105E (HW v5 & v5.6)
- TP-Link TL-SG105PE (HW v2.60)
- TP-Link TL-SG108E (HW v6 & v6.6)
- TP-Link TL-SG108PE (HW v4, v4.20, v4.60, v5, & v5.60; possibly v5.46)
- TP-Link TL-SG116E (HW v2.20 & v2.26)
- TP-Link TL-SG1016PE (HW v5, v5.20, v5.26, & v5.60; possibly v6, v6.6)
- TP-Link TL-SG1024DE (HW v4.20, 4.26, v6, v6.6, v7, & v7.60)
- TP-Link TL-SG1428PE (HW v2, v2.20, v2.60, v3, v3.60)
- Other TP-Link Easy Smart Switch models with compatible firmware (for example v1.0.0 build 20230214 or newer where applicable) may work if they use the same web management UI.
- TP-Link Easy Smart Switch hardware may require firmware **v1.0.0 build 20230214** or later
