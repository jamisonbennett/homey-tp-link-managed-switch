Strømlinjeform nettverket ditt med smart automatisering: koble TP-Link Easy Smart Switch-enheter til Homey for portstyring, status og flytbasert nettverksautomatisering.

Funksjoner:
- Portstyring: aktiver eller deaktiver enkeltporter på TP-Link Easy Smart Switch-enheten direkte fra Homey-appen eller via Homey Flows.
- Enhetsintegrasjon: sømløs integrasjon med flere TP-Link Easy Smart Switch-modeller.
- Automatisering: opprett Homey Flows for å automatisere nettverksadministrasjon basert på dine preferanser eller bestemte betingelser—inkludert LED-styring, reaksjon når koblingsstatus endres, og omstart av switchen ved behov.
- Statusovervåking: se status for hver port (på/av) i Homey-appen.
- Oppdagelse av koblingsendringer: switchporter overvåkes for koblingsstatus omtrent hvert minutt.

Switchen må være på samme lokale nettverk som Homey-enheten din. Bruk fast IP-adresse eller DHCP-reservasjon slik at adressen ikke endres. Oppgi samme brukernavn og passord som du bruker på administrasjonssiden til switchen på nett.
