Rationalisez votre réseau grâce à l’automatisation intelligente : connectez les commutateurs TP-Link Easy Smart Switch à Homey pour le contrôle des ports, l’état et l’automatisation réseau pilotée par les Flows.

Fonctionnalités :
- Contrôle des ports : activez ou désactivez chaque port de votre commutateur TP-Link Easy Smart Switch depuis l’application Homey ou via les Flows Homey.
- Intégration des appareils : intégration fluide avec plusieurs modèles TP-Link Easy Smart Switch.
- Automatisation : créez des Flows Homey pour automatiser la gestion du réseau selon vos préférences ou des conditions précises—y compris le contrôle des LED, les réactions aux changements d’état de liaison et le redémarrage du commutateur si nécessaire.
- Surveillance de l’état : consultez l’état de chaque port (activé/désactivé) dans l’application Homey.
- Détection des changements de liaison : les ports du commutateur sont surveillés environ une fois par minute.

Le commutateur doit être sur le même réseau local que votre Homey. Utilisez une adresse IP fixe ou une réservation DHCP pour que l’adresse ne change pas. Saisissez le même nom d’utilisateur et le même mot de passe que pour la page web d’administration du commutateur.

Matériel pris en charge :

- TP-Link TL-SG105E (HW v5 & v5.6)
- TP-Link TL-SG105PE (HW v2.60)
- TP-Link TL-SG108E (HW v6 & v6.6)
- TP-Link TL-SG108PE (HW v4, v4.20, v4.60, v5 & v5.60 ; éventuellement v5.46)
- TP-Link TL-SG116E (HW v2.20 & v2.26)
- TP-Link TL-SG1016PE (HW v5, v5.20, v5.26 & v5.60 ; éventuellement v6, v6.6)
- TP-Link TL-SG1024DE (HW v4.20, 4.26, v6, v6.6, v7 & v7.60)
- TP-Link TL-SG1428PE (HW v2, v2.20, v2.60, v3, v3.60)
- D’autres modèles TP-Link Easy Smart Switch avec un firmware compatible (par exemple v1.0.0 build 20230214 ou plus récent le cas échéant) peuvent fonctionner s’ils utilisent la même interface web d’administration.
- Le matériel TP-Link Easy Smart Switch peut nécessiter le firmware **v1.0.0 build 20230214** ou une version ultérieure
