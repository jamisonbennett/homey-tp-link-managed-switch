스마트 자동화로 네트워크를 간소화하세요: TP-Link Easy Smart 스위치를 Homey에 연결하여 포트 제어, 상태 확인, Flow 기반 네트워크 자동화를 사용할 수 있습니다.

기능:
- 포트 제어: Homey 앱 또는 Homey Flow에서 TP-Link Easy Smart 스위치의 개별 포트를 켜거나 끕니다.
- 기기 연동: 여러 TP-Link Easy Smart 스위치 모델과 원활하게 연동합니다.
- 자동화: 선호 조건이나 특정 상황에 맞춰 Homey Flow로 네트워크 관리를 자동화합니다—LED 제어, 링크 상태 변경에 대한 반응, 필요 시 스위치 재시작을 포함합니다.
- 상태 모니터링: Homey 앱에서 각 포트의 상태(켜짐/꺼짐)를 확인합니다.
- 링크 변경 감지: 스위치 포트의 링크 상태를 약 1분마다 확인합니다.

스위치는 Homey와 같은 로컬 네트워크에 있어야 합니다. 주소가 바뀌지 않도록 고정 IP 또는 DHCP 예약을 사용하세요. 스위치 웹 관리 페이지와 동일한 사용자 이름과 비밀번호를 입력하세요.
