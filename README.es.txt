Optimiza tu red con automatización inteligente: conecta los conmutadores TP-Link Easy Smart Switch a Homey para control de puertos, estado y automatización de red mediante Flows.

Funciones:
- Control de puertos: activa o desactiva puertos individuales en tu TP-Link Easy Smart Switch desde la app Homey o mediante Flows de Homey.
- Integración de dispositivos: integración fluida con varios modelos TP-Link Easy Smart Switch.
- Automatización: crea Flows de Homey para automatizar la gestión de la red según tus preferencias o condiciones concretas—incluido el control del LED, reaccionar a cambios del estado del enlace y reiniciar el conmutador cuando sea necesario.
- Monitorización del estado: consulta el estado de cada puerto (activado/desactivado) en la app Homey.
- Detección de cambios de enlace: los puertos del conmutador se supervisan aproximadamente cada minuto.

El conmutador debe estar en la misma red local que tu Homey. Usa una dirección IP fija o una reserva DHCP para que la dirección no cambie. Introduce el mismo usuario y contraseña que uses en la página web de administración del conmutador.

Hardware compatible:

- TP-Link TL-SG105E (HW v5 y v5.6)
- TP-Link TL-SG105PE (HW v2.60)
- TP-Link TL-SG108E (HW v6 y v6.6)
- TP-Link TL-SG108PE (HW v4, v4.20, v4.60, v5 y v5.60; posiblemente v5.46)
- TP-Link TL-SG116E (HW v2.20 y v2.26)
- TP-Link TL-SG1016PE (HW v5, v5.20, v5.26 y v5.60; posiblemente v6, v6.6)
- TP-Link TL-SG1024DE (HW v4.20, 4.26, v6, v6.6, v7 y v7.60)
- TP-Link TL-SG1428PE (HW v2, v2.20, v2.60, v3, v3.60)
- Otros modelos TP-Link Easy Smart Switch con firmware compatible (por ejemplo v1.0.0 build 20230214 o más reciente cuando aplique) pueden funcionar si usan la misma interfaz web de administración.
- El hardware TP-Link Easy Smart Switch puede requerir firmware **v1.0.0 build 20230214** o posterior
