Effektivisera ditt nätverk med smart automatisering: anslut TP-Link Easy Smart Switch-enheter till Homey för portstyrning, status och flödesdriven nätverksautomatisering.

Funktioner:
- Portstyrning: aktivera eller inaktivera enskilda portar på din TP-Link Easy Smart Switch direkt från Homey-appen eller via Homey Flows.
- Enhetsintegration: sömlös integration med flera TP-Link Easy Smart Switch-modeller.
- Automatisering: skapa Homey Flows för att automatisera nätverkshantering utifrån dina önskemål eller specifika villkor—inklusive LED-styrning, reaktion när länkstatus ändras och omstart av switchen vid behov.
- Statusövervakning: se status för varje port (på/av) i Homey-appen.
- Detektering av länkändringar: switchportar övervakas för länkstatus ungefär en gång per minut.

Switchen måste finnas på samma lokala nätverk som din Homey. Använd en fast IP-adress eller DHCP-reservation så att adressen inte ändras. Ange samma användarnamn och lösenord som för switchens webbhanteringssida.
